## 一、线性代数（✅ 已上线，2026-07-19，5 讲全部完成）

**课程主线**：从"解方程组的记账技巧"到"向量空间的几何语言"——抽象是一层层被逼出来的。
**总困境**：教科书上来就是行列式和矩阵定义，学生不知道这些运算规则为什么长这样。

### 第 1 讲：线性方程组与消元法
- 困境：《九章算术》方程术（公元前 1 世纪！）与高斯测量大地——多元一次方程组的系统解法
- 关键：把未知数抽走，只对系数表操作 → "矩阵"最初就是一张记账表
- 动画：2×2 / 3×3 方程组的消元过程逐步可视化；每步同时显示几何图（两直线/三平面的交点如何不变）
- 联系：计算数学的 LU 分解就是消元的矩阵化记录

### 第 2 讲：行列式
- 困境：方程组何时无解/多解？克莱姆（1750）发现判别式；莱布尼茨更早就在信里写过
- 关键直觉：**行列式 = 有向面积/体积的缩放因子**（而不是那个展开式）
- 动画：拖动 2×2 矩阵的两个列向量，实时显示平行四边形面积 = det；det 变负时翻转、det=0 时压扁成线（这正是"奇异"的含义）
- 联系：积分换元的雅可比行列式、叉积

### 第 3 讲：矩阵与线性变换
- 困境：为什么矩阵乘法规则如此古怪（行乘列）？凯莱 1858《矩阵论回忆录》
- 关键：矩阵不是数表，是**线性变换的名片**；乘法古怪是因为它就是"变换复合"
- 动画：单位网格在拖动矩阵参数下变形（旋转/剪切/伸缩）；两个变换先后作用 = 乘积矩阵一步作用
- 联系：导数讲里的"局部线性近似"——多元函数的导数就是一个矩阵（雅可比）

### 第 4 讲：向量空间与线性无关
- 困境：函数、多项式、数列怎么也能"相加、数乘"？1888 皮亚诺公理化
- 关键：抽象不是故弄玄虚——同一套定理一次证明，处处可用（这是抽象的"买到了什么"）
- 动画：三个三维向量的张成空间：拖动向量，看张成从线→面→全空间的跳变；线性相关的瞬间塌缩
- 联系：傅里叶级数就是"函数空间里的正交坐标系"（为数学物理方程铺路）

### 第 5 讲：特征值与特征向量
- 困境：变换那么复杂，有没有"不改方向只改长度"的特殊方向？
- 历史：源于振动问题和天体力学的主轴（拉格朗日、柯西）
- 动画：拖动任意向量 v，同屏画 Av；当 v 扫过特征方向时 Av 与 v 共线，高亮闪现；配斐波那契/人口迁移的幂迭代演示"特征值决定长期命运"
- 联系：微分方程组解耦、主成分分析、谷歌 PageRank；二次型主轴 → 概率论的协方差矩阵

## 二、概率论（✅ 已上线，2026-07-19，5 讲全部完成）

**课程主线**：从赌桌纠纷到公理化——"运气"如何一步步变成可计算的数学。
**用户背景**：学过概率论，重点是把测度式定义（分布函数、期望、大数定律）讲出动机。

### 第 1 讲：概率的诞生——点数分配问题
- 困境：1654 年赌徒问帕斯卡：赌局中途中断，赌注怎么分才公平？帕斯卡-费马通信
- 关键：不看已发生的，看**未来所有等可能结局的比例**——概率的古典定义由此而来
- 动画：中断赌局的树状图逐层展开，数出有利结局占比
- 陷阱案例：达朗贝尔的错误（两枚硬币"三种结果"）——等可能不能拍脑袋

### 第 2 讲：条件概率与贝叶斯
- 困境：拿到新信息后，"可能性"怎么更新？贝叶斯 1763（死后发表）
- 动画：经典疾病检测问题的面积图——1000 人方阵按患病/检测结果着色，直观看出"阳性但未患病"为何占多数
- 联系：现代机器学习、垃圾邮件过滤

### 第 3 讲：随机变量与分布——从掷骰子到函数
- 困境：为什么要把"结果"变成"数"？连续情形单点概率为 0，怎么谈概率？
- 关键：密度不是概率，是概率的"变化率"——**分布函数 F 才是本体，密度是它的导数**（微积分基本定理再现！）
- 动画：直方图组距逐步细化 → 极限成密度曲线；面积 = 概率
- 联系：积分讲里"概率是密度曲线下面积"的伏笔在此兑现

### 第 4 讲：期望、方差与大数定律
- 困境：惠更斯 1657"赌局的价值"；伯努利 1713《猜度术》——频率凭什么趋于概率？
- 动画：模拟掷骰子 n 次的样本均值轨迹，n 拖到 10000，多条轨迹收敛入 ε 带（与极限讲的 ε 带呼应！）
- 圣彼得堡悖论：期望无穷的游戏你愿出多少钱？——期望不是全部

### 第 5 讲：中心极限定理与正态分布
- 困境：为什么误差、身高、噪声都长成钟形？棣莫弗 1733 → 拉普拉斯 → 高斯
- 动画：高尔顿板小球下落堆成钟形；任意奇形怪状的原始分布，取样本和，n 增大分布形状趋于正态
- 联系：泰勒展开（棣莫弗用它逼近二项系数）；测量误差 → 最小二乘（线性代数）

## 三、复变函数（✅ 已上线，2026-07-19，5 讲全部完成）

**课程主线**：√−1 从"不可能的数"到分析学最优雅的分支——三百年的接纳史。
**总困境**：学生会算留数但不知道为什么复可导如此强大、积分为何只看奇点。

### 第 1 讲：虚数的诞生——三次方程的倒逼
- 困境：卡尔达诺 1545：解三次方程时，**实数解**的求解过程中途必须经过 √负数（不可约情形）——不接受它就得不到明明存在的实根
- 关键：虚数不是为了解 x²+1=0 发明的（那可以耸耸肩说无解），是被三次方程逼出来的
- 动画：邦贝利的例子 x³=15x+4：中途出现 √−121，最后神奇约掉得 x=4
- 复平面：韦塞尔/阿尔冈/高斯——乘 i = 旋转 90°，虚数从此"看得见"

### 第 2 讲：复函数与复可导——一个苛刻得惊人的条件
- 困境：把导数定义照搬到复数，看似平常，实则天翻地覆——极限要求**从任何方向逼近都相等**
- 关键：柯西-黎曼方程；复可导 ⇒ 无穷次可导 ⇒ 解析（实函数完全没有的奇迹）
- 动画:复函数看作平面到平面的映射：网格在 z²、eᶻ 下的变形；解析函数处处保角（小方格映成小方格），|z|² 这类不解析的立刻扭曲失角
- 联系：线性代数第 3 讲——复可导 = 局部变换恰好是"旋转+伸缩"矩阵

### 第 3 讲：柯西积分定理与积分公式
- 困境：复平面上从 A 到 B 有无数条路径，积分值凭什么不依赖路径？
- 关键：只要区域内解析，回路积分为零（柯西 1825）；于是函数在圆心的值由圆周上的值完全决定——解析函数是"全息"的
- 动画：拖动积分路径变形，实时数值积分显示值不变；路径圈住奇点的瞬间值跳变
- 联系：微积分基本定理的家族成员（"内部累积=边界取值"）

### 第 4 讲：留数定理——把积分变成数圈圈
- 困境：实积分 ∫dx/(1+x⁴) 之类硬算极难，物理里却到处是
- 关键：回路积分 = 2πi ×（圈内各奇点留数之和）——无穷多项的积分信息浓缩在几个点上
- 动画：选不同回路圈住不同奇点组合，积分值 = 所圈留数和的实时验证
- 联系：泰勒展开讲埋的伏笔在此收割——**收敛半径 = 到最近奇点的距离**，"半径"一词的谜底

### 第 5 讲：解析延拓与黎曼的遗产（选做/短讲）
- ζ 函数与素数：解析函数"一小段决定全体"的极致应用
- 联系：为数学物理方程的调和函数（拉普拉斯方程）铺路——解析函数的实部虚部都是调和函数

## 四、数学物理方程（✅ 已上线，2026-07-19，5 讲全部完成）

**课程主线**：三大方程 = 三种物理图景；解法（分离变量、傅里叶）= 线性代数在函数空间的胜利。
**特别注意**：用户没学过这门课，每讲要从物理直觉零起点铺垫，公式推导放慢，
偏导数、场的概念先用一讲补齐。

### 第 0 讲：预备——从常微分到偏微分
- 困境：单摆一个未知函数 x(t)；但弦上每一点都在动——未知量是 u(x,t)，一个"函数的函数场"
- 补齐：偏导数的直观（固定一个变量切片看）、什么是"场"
- 动画：u(x,t) 曲面：沿 t 切是某点的振动史，沿 x 切是某时刻的弦形

### 第 1 讲：波动方程——弦振动之争
- 困境：1746 达朗贝尔导出 uₜₜ = c²uₓₓ；欧拉、丹尼尔·伯努利、拉格朗日为"任意初始形状都能表示成正弦叠加吗"吵了半个世纪
- 推导：微元受力分析（张力的垂直分量差 → 二阶偏导）——只用牛顿第二定律
- 动画：拨一下弦，行波分解 u = f(x−ct) + g(x+ct) 左右跑；驻波模式（基频、泛音）叠加出任意形状
- 联系:音乐（泛音列）、达朗贝尔行波解

### 第 2 讲：热传导方程与傅里叶级数
- 困境：1807 傅里叶的论文被拉格朗日拒稿——"任意函数都能展成三角级数"太离经叛道
- 推导：热量守恒 + 牛顿冷却 → uₜ = a²uₓₓ；为何一阶时间导数（不可逆！与波动方程本质不同）
- 动画：方波的傅里叶部分和逐项叠加（吉布斯现象可见）；初始温度分布在热方程下各频率分量按 e^{−k²t} 衰减——高频先死，分布变平滑
- 联系：线性代数第 4 讲的"函数空间正交基"在此兑现；泰勒展开是"用多项式冒充"，傅里叶是"用正弦波冒充"——两大逼近哲学对照

### 第 3 讲：拉普拉斯方程——稳态与调和
- 困境：引力势（拉普拉斯《天体力学》）、静电场、稳态温度都满足 Δu = 0——为什么殊途同归？
- 关键直觉：调和函数 = "每点的值等于邻域平均值"——没有局部凸起的最平滑状态
- 动画：正方形区域给定边界温度，松弛法迭代收敛到稳态（每点不断取邻居平均），可拖动边界条件
- 联系：复变函数第 5 讲——解析函数实部就是调和函数；平均值性质 ↔ 柯西积分公式

### 第 4 讲：分离变量法——三大方程的统一解法
- 关键：猜 u = X(x)T(t) → 偏微分方程劈成两个常微分方程 + 特征值问题
- 本质：**求解 = 在正弦基下把问题对角化**——与线性代数第 5 讲的特征值解耦是同一件事
- 动画：任意初始条件 → 分解到各特征模式 → 每个模式按各自规律演化（波动：振荡；热：衰减）→ 叠加回来
- 收官：三大方程对照表（双曲/抛物/椭圆），整个网站的概念在此大会师

## 五、数值分析（✅ 已上线，2026-07-19，5 讲全部完成）

**课程主线**：当纯数学的"精确解"失效（无求根公式、无初等原函数、机器存不准），
如何用有限步、有限精度算出误差可控、够用的近似——两条暗线贯穿全课：**截断误差 / 舍入误差**，
一位英雄：**迭代逼近 + 误差分析**。面向"学得晕乎乎"的初学者，每讲抓一个教科书级经典例子。
**用户背景**：把数值分析当作用户学过、但只会套公式的课来讲动机；勾连尽量用站内 router-link 兑现伏笔。

### 第 1 讲：浮点数与两种误差
- 困境：0.1+0.2≠0.3；爱国者导弹计时漂移、奔腾 FDIV、阿丽亚娜 5 号——舍入误差酿的祸
- 关键：机器精度 eps=2⁻⁵²；**灾难性抵消**（相近数相减吃掉有效数字）才是真凶
- 动画（CancellationDemo）：真用 float64 算 (1−cos x)/x²，x→0 时红线从 0.5 当场崩坏；改写成 2sin²(x/2)/x² 稳如泰山
- 联系：截断误差鼻祖 = 泰勒余项；为第 5 讲条件数（放大器）埋首尾呼应的伏笔

### 第 2 讲：方程求根——二分法与牛顿法
- 困境：阿贝尔证五次方程无求根公式；超越方程更无解析解，工程却要具体数
- 关键：二分法稳但线性收敛；**牛顿法用切线（一阶泰勒）二次收敛**，正确位数翻倍；巴比伦开方 = 牛顿法
- 动画（NewtonMethodDemo）：拖 x₀ 与步数，切线楼梯滑向 √2，读数看误差 0.1→0.01→1e-4→1e-8
- 联系：多元牛顿每步解 JᐧΔx=−f（雅可比→线性代数、第 5 讲）；复变的牛顿分形

### 第 3 讲：插值与龙格现象
- 困境：只有离散表，要中间值 → 拉格朗日插值；直觉"点越多次数越高越准"
- 关键：**龙格 1901 反例** 1/(1+25x²) 等距插值两端爆震；深层原因在复平面 ±i/5 的极点（收敛半径=到最近奇点距离，复变第 4 讲收割）；解药：切比雪夫点 / 样条
- 动画（RungeDemo）：拖节点数看等距插值两端发散，一键切切比雪夫点平息
- 联系：范德蒙线性方程组病态（→第 5 讲）；插值余项 ↔ 泰勒余项孪生

### 第 4 讲：数值积分——从数格子到高斯求积
- 困境：∫e^{−x²} 无初等原函数（刘维尔），微积分基本定理失效，正态分布表却要算
- 关键：回到面积；矩形/梯形 O(h²)，**辛普森用抛物线 O(h⁴)、代数精度越级到 3**；理查森外推/龙贝格；高斯求积把取点位置也优化，n 点精确 2n−1 次
- 动画（QuadratureDemo）：切换中点/梯形/辛普森，拖 N 对比误差数量级（h² vs h⁴）
- 联系：黎曼和升级版、基本定理适用边界；蒙特卡洛 = 大数定律/CLT（概率论）；牛顿-柯特斯高阶也犯龙格 → 复合辛普森

### 第 5 讲：线性方程组的数值解——主元与条件数
- 困境：手算消元得精确分数，浮点下小主元放大误差、希尔伯特矩阵病态——同一方程组算出天壤之别
- 关键：部分主元选取（威尔金森后向误差分析）；**条件数 κ(A)=σmax/σmin（图灵 1948）= 误差放大器**；两条近平行直线求交点 = 病态直观
- 动画（ConditionNumberDemo）：拖分离度把良态拧成病态，微扰 δ 让交点乱飞，实时算 κ
- 联系：消元=LU、det≈0↔病态、迭代收敛由谱半径（线性代数全套武器被重新称量）；与第 1 讲首尾呼应——κ 正是放大渺小舍入误差的那只手；岭回归=压条件数
